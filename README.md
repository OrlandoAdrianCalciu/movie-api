# myFlix-app
## Movie API

### Description:
Creating a backend API to store information about movies and allow users to create lists of favorite movies.

### Dependencies
- Node.JS
- express
- express-validator
- passport
- passport-jwt
- passport-local
- body-parser
- mongoose
- cors
- bcrypt
- morgan
- lodash
- Executing program

For a list of endpoints and responses check out the documentation.html
