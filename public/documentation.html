Documentation

The web application will provide users with acces to information about different movies, directors and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Businnes Logic URL HTTP Method Request body data format Response body data format
Get a list of all movies /movies GET None A JSON object holding data about all the students
Get data about a single movie /movies/:Title GET None A JSON object holding data about a single movie, containing the title, author, genre and a image. Example: 
                        
                        {
                            title: 'The Fate of the Furious',
                            author: {
                                name: 'Chris Morgan',
                                born: 'December 5, 1970'
                            },
                            genre: {
                                name: 'Action',
                                description: 'A film with a fast-moving plot, usually containing scenes of violence.'
                            },
                            img: '',
                        }
Get data about a genre /movies/genres/:Name GET None A JSON object holding data about the genre
Get data about an author /movies/director/:Name GET None A JSON object holding data about an author
Create a user /users POST A JSON object holding data about the user to add: { name: "Max", favoriteMovies: ["The Fate of the Furious"] } A JSON object holding data about the user that was added: 
                        
                        {
                            id: 1,
                            name: "Max",
                            favoriteMovies: ["The Fate of the Furious"]
                        }
Update a user /users/:Username PUT None A text message indicating the user
Add a movie to their favorites /users/:Username/movies/:MovieID POST None A text message indicating that the movie has been added to favorites
Delete a movie from their favorites /users/:Username/movies/:MovieID DELETE None A text message indicating that the movie has been removed
Delete the existing user /users/:Username DELETE None A text message indicating that the user was deleted
